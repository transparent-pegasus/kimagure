# きまぐれごはん v1.0 リリースノート

「計画しない」が、いちばん続く。
その日の気分と食べたものを入力するだけで、「今日を100点にする献立」を提案するアプリです。

---

## 主な機能

### 1. 食事プランの入力

朝・昼・夜の3枠に、食べたもの・食べる予定のものを自由に入力できます。間食や夜食などの枠を追加することも可能です。

入力欄を「提案」に切り替えると、その食事の献立をAIが考えてくれます。

### 2. AI による献立提案

入力された食事記録をもとに、管理栄養士の視点で1日の栄養バランスを調整した献立を提案します。

- 既に食べたものの栄養素を推定し、残りの枠で足りない栄養を補うメニューを生成
- 生成した献立には「なぜこのメニューを選んだか」の理由が付与される
- 各料理にはYouTube検索リンクが付いているため、調理方法をすぐに確認できる

### 3. リクエスト機能

使いたい材料や食べたいジャンルを指定して、提案内容を好みに合わせることができます。

### 4. ボリューム・構成の選択

提案される献立の規模を4段階から選べます。

- 一汁三菜: しっかり定食
- 一品料理: 丼・麺など
- おかずのみ: 主食なし
- 軽食: サラダ・スープなど

### 5. 条件設定

調理の手間を「手作り」「一部既製品」「スーパー・コンビニ」「外食」から選択できます。

栄養バランスの調整単位を「日単位」「週単位」「無視」から選べます。

生年と性別を登録すると、摂取カロリーの目安が自動表示されます。

### 6. 食材・栄養制限

糖質、砂糖、油、食塩相当量などの制限を設定できます。任意の食材・栄養を追加して、完全にNG、または上限値付きの制限として登録することも可能です。

### 7. 履歴管理

保存した献立は過去30日分を一覧表示し、いつでも振り返ることができます。
